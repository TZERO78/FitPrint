OpenMailPrint

A free, open-source Microsoft Outlook add-in that prints the selected email cleanly and automatically scales oversized images down to the page width — so they are no longer cut off when you print.

The problem

Modern Outlook renders messages with the Word engine and no longer offers a “shrink to fit page” option. As a result, wide or high-resolution images get clipped at the right edge when you print an email, and there is no built-in way to fit them to the page.

What OpenMailPrint does

  • Reads the selected message and embeds its inline images.
  • Scales oversized images down to the page width and corrects their EXIF rotation.
  • Adds a From / To / Cc / Date / Subject header and opens the normal print dialog.
  • Only ever reads the message — it never modifies your email.

Install

It is hosted on GitHub Pages, so there is nothing to build. Sideload the manifest once:

https://tzero78.github.io/OpenMailPrint/manifest.xml

See the install instructions for work/school accounts and for personal Outlook.com accounts.

Compatibility

OpenMailPrint is an Office web add-in, so it runs in Outlook on the web, new and classic Outlook for Windows, and Outlook for Mac — wherever the platform supports Office web add-ins (Mailbox requirement set 1.8).
